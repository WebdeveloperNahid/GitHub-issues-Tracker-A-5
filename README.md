1. Difference between var, let, and const:

* var = Function scoped and can be re-declared & updated.
* let = Block scoped and can be updated but not re-declared.
* const = Block scoped and cannot be updated or re-redlared.


2.What is the spread operator (...)?

*The spread operator is used to expand or copy elements of an array or object.


3.What is the difference between map(), filter(), and forEach()?

* filter() = Creates a new array by modifying each element.
* filter() = Creates a new array with elements that match a condition.
* forEach() = Loops through array elements but does not return a new array.

4.What is an arrow function?

*An arrow function is a shorter syntax for writing functions using =>.

5.What are template literals?

*Template literals are strings written with backticks ( ) that allow embedded variables using ${}.
